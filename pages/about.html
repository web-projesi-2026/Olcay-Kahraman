Ben Kimim? Geleceği Kodluyorum.

Merhaba! Ben Olcay KAHRAMAN. Bir üniversite öğrencisi ve Full-Stack Geliştirici adayıyım. Yazılım dünyasına adım attığım günden beri, karmaşık problemleri basit ve kullanıcı dostu arayüzlere dönüştürmeyi hedefliyorum.

Backend tarafında C# ve Java ile veriyi yönetirken, Frontend dünyasında JavaScript'in dinamizmiyle modern deneyimler tasarlıyorum. Teknolojiyi sadece bir araç değil, bir çözüm sanatı olarak görüyorum.

Nelerle Uğraşıyorum?

Backend

ASP.NET Core, SQL Server, Java Swing, File I/O işlemleri üzerine yoğunlaşıyorum.

Frontend

Modern HTML5, CSS3 ve JavaScript kullanarak interaktif kullanıcı deneyimleri oluşturuyorum.

Mobile

Flutter ve Dart ile mobil uygulama geliştirme süreçlerine hakimim.
