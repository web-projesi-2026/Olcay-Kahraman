# Olcay KAHRAMAN — Kişisel Portfolyo Sitesi
## Proje Açıklaması
Olcay KAHRAMAN'ın Full-Stack geliştirici kimliğini yansıtan, modern, performans odaklı ve kullanıcı dostu bir web uygulamasıdır. Bu proje, sadece bir tanıtım sitesi değil; arka planda ASP.NET Core mimarisi ve MS SQL Server ile güçlendirilmiş dinamik bir yönetim sistemidir.

# Hedef Kullanıcı
İşverenler ve İK Uzmanları: Yeteneklerimi, projelerimi ve özgeçmişimi incelemek isteyen profesyoneller.
İş Ortakları: Teknik yetkinliklerimi analiz edip iş birliği yapmak isteyen geliştiriciler.
Ziyaretçiler: İletişim formu üzerinden geri bildirim veya teklif göndermek isteyen kullanıcılar.

# Temel Özellikler

Dinamik Yönetim: Tüm içeriklerin (projeler, özgeçmiş, hakkımda) admin paneli üzerinden güncellenebilir olması.
Modern Arayüz: Karanlık tema (Dark Mode) desteği, akıcı animasyonlar ve şık UI/UX tasarımı.
Dashboard: Mesajların ve site istatistiklerinin takip edilebildiği gelişmiş admin paneli.
Özgeçmiş Yönetimi: Eğitim ve deneyim bilgilerinin kronolojik olarak sergilenmesi.
Tam Responsive: Mobil, tablet ve masaüstü cihazlarla %100 uyumlu tasarım.
İletişim Sistemi: Doğrudan veritabanına kayıt yapan ve bildirim odaklı iletişim formu.
